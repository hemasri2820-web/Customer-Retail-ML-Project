# Customer Retail Machine Learning Project

## Project Objective
Compare multiple machine learning models using customer retail data.

## Features Used
- Quantity
- UnitPrice
- Country

## Models Implemented
- Logistic Regression
- Decision Tree
- K-Nearest Neighbors (KNN)
- Random Forest
- Support Vector Machine (SVM)

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Results
Compared model performance using accuracy and evaluation metrics.
